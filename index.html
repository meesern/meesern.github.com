I Only Wanted a Simple Blog

I suppose that it is quite typical. I set out to start a simple blog to write about the many many interesting things that I get involved with. Simple - go to wordpress.com and voilà! A Blog! Great, but I was going to be putting quite a number of code excerpts into the posts and wanted to be sure that syntax highlighting in multiple languages would work. It’s easy with a wordpress plugin but less so on wordpress.com as it does not support custom css/html in the template. According to this great guide to syntax highlighting in blogger Blogger does support editing the template HTML and so I created a blog there as well to compare the two.
